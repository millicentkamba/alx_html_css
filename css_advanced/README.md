Project: CSS Advanced

This project extends the HTML Advanced work. You focus on layout, spacing, color, and typography. You use the Figma file as your guide.

Your goals
• Recreate the full design with clean CSS.
• Match the structure of the HTML Advanced page.
• Replace placeholders with real images.
• Follow the spacing and sizing in the design.
• Keep your HTML simple and readable.
• Keep your CSS organized and consistent.

What you do
• Download the images from the project link.
• Duplicate the Figma file to your drafts.
• Inspect sizes, margins, colors, and text styles.
• Update the HTML where images or classes are missing.
• Add all assets to an images folder.
• Link a styles.css file.
• Test the page in a browser.
